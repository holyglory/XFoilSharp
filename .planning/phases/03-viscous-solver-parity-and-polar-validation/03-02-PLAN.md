       ComputeCriticalReynoldsNumber(double hk) -> double
       Port the Falkner-Skan log(Re_crit) vs Hk correlation from xblsys.f.

    2. **DAMPL** (xblsys.f:1840-1950) -- standard amplification rate for Hk < 3.5:
       ComputeAmplificationRate(double hk, double theta, double ue, double mSquared, double reReference)
       -> (double Ax, double Ax_Hk, double Ax_Th, double Ax_Rt)

       Key details:
       - Computes local Re_theta = ue * theta / nu (from Re reference)
       - Gets critical Re from Falkner-Skan correlation
       - Smooth onset ramp: RFAC = (log(Re_theta) - log(Re_crit)) / DGR where DGR = 0.08
       - Below onset (RFAC < 0): AX = 0 with zero sensitivities
       - In ramp (0 <= RFAC <= 3): RFAC smoothly ramps from 0 to 1
       - Full amplification (RFAC > 3): AX from Drela's spatial amplification correlation
       - Returns AX and sensitivities AX_HK, AX_TH, AX_RT

       ComputeAmplificationRateHighHk(double hk, double theta, double ue, double mSquared, double reReference)
       -> (double Ax, double Ax_Hk, double Ax_Th, double Ax_Rt)

       Uses simplified growth rate correlation valid for highly separated BLs.

    4. **AXSET** (xblsys.f:2040-2160) -- combined amplification rate dispatch:
       ComputeTransitionSensitivities(double hk, double theta, double ue, double mSquared, double reReference, double hStar, double hStar_Hk, double hStar_Rt, double hStar_Msq)
