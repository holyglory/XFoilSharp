<objective>
Port the BL equation system assembly routines (BLSYS, BLDIF, TRDIF, BLMID, TESYS, BLPRV, BLVAR) from xblsys.f. These assemble the 3-equation BL residuals and their Jacobian blocks for each station pair, which SETBL will later use to build the global Newton system.

Purpose: The BL system assembly is the innermost computational kernel of the viscous solver. It computes the momentum, shape parameter, and amplification/Ctau residuals and all their sensitivities for each BL station pair. Without correct assembly, the Newton system will not converge.

Output: BoundaryLayerSystemAssembler static class with all assembly functions, tests.
</objective>

    Continue in BoundaryLayerSystemAssembler.cs:

    5. **BLDIF** (xblsys.f:700-1100): ComputeFiniteDifferences -- the core 3-equation finite-difference residual. Given BL variables at stations 1 and 2 and their midpoint correlations (from BLMID), computes:
       - Momentum integral equation residual (von Karman)
       - Shape parameter equation residual (kinetic energy or entrainment)
       - Shear stress equation residual (Ctau equation for turbulent) or amplification (for laminar)
       Returns residual[3] and Jacobian blocks A[3,5] (station 2 sensitivities) and B[3,5] (station 1 sensitivities).

       The 5 columns per station are: d/d(mass defect), d/d(theta), d/d(Ctau or N), d/d(Ue), d/d(xi).
       These get mapped to the 3-variable system (mass defect, theta, Ctau) plus edge velocity coupling in SETBL.

    6. **TRDIF** (xblsys.f:1100-1500): ComputeTransitionDifferences -- modified BLDIF for the interval containing the transition point. The third equation becomes the amplification constraint: N2 - N1 - integral(dN/ds * ds) = 0. Jacobian includes sensitivities from AXSET. This is only used at the transition interval.

    7. **TESYS** (xblsys.f:600-700): AssembleTESystem -- special assembly for the TE-to-wake junction point. The first wake station inherits TE boundary conditions: theta_wake = theta_upper + theta_lower, delta*_wake = delta*_upper + delta*_lower. The Jacobian VZ[3,2] block couples the two surface sides at TE.

    8. **BLSYS** (xblsys.f:1-126): AssembleStationSystem -- top-level dispatcher that:
       - Calls BLPRV for compressibility at both stations
       - Calls BLKIN + BLVAR for correlations at both stations
       - Calls BLMID for midpoint correlations
       - Calls BLDIF (or TRDIF if transition interval) for residuals
       - Maps the 5-column Jacobian to the 3-column (mass, theta, Ctau) VA/VB blocks
       - Stores results in the Newton system arrays

    All functions: port Fortran exactly preserving operation order and Jacobian chain rules. The mapping from 5-column sensitivities to 3-column system blocks (dropping Ue and xi, which get absorbed via DIJ coupling and arc-length geometry) is critical.
