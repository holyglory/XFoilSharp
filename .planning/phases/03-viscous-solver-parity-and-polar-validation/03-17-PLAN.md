<interfaces>
BoundaryLayerCorrelations already returns derivatives that BLDIF needs. Each correlation method returns a tuple with _hk, _rt, _msq suffixed derivatives. For example:

- SkinFriction(hk, rt, msq) returns (cf, cf_hk, cf_rt, cf_msq)
- LaminarDissipation(hk, rt) returns (di, di_hk, di_rt)
- TurbulentDissipation(hk, rt, msq, hs) returns (di, di_hk, di_rt, di_msq, di_hs)
- KineticEnergyThickness (HSL/HST) returns (hs, hs_hk, hs_rt, hs_msq)
- DensityThicknessShape (HCT) returns (hc, hc_hk, hc_msq)

These _hk, _rt, _msq derivatives correspond to Fortran's CF2_HK2, CF2_RT2, CF2_M2 etc. The chain rule then extends them to _T1, _D1, _U1 via:
  CF2_T2 = CF2_HK2*HK2_T2 + CF2_RT2*RT2_T2
  CF2_D2 = CF2_HK2*HK2_D2
  CF2_U2 = CF2_HK2*HK2_U2 + CF2_RT2*RT2_U2 + CF2_M2*M2_U2

The HK, RT, M2 primary derivatives are computed from:
  HK = (D/T - 1) * compressibility factor  -->  HK_T, HK_D, HK_U, HK_MS
  RT = reybl * U * T                        -->  RT_T, RT_U, RT_RE
  M2 = U^2 * MSQ_factor                     -->  M2_U, M2_MS

These primary derivative chains (HK_T1, HK_D1, RT_T1, RT_U1, M2_U1, M2_MS) need to be computed inside ComputeFiniteDifferences (matching Fortran BLVAR), then used to chain all correlation derivatives to the 5 Newton variables (S, T, D, U, X).

**Step 1: Compute BLVAR-style primary derivatives (Fortran BLVAR lines 784-870)**

At both stations (1 and 2), compute the primary derivative chains from BL variables to Newton variables. These are NOT currently computed anywhere in C#:

```
// Station 2 primary derivatives (same pattern for station 1)
// H = D/T, so:
double h2_t2 = -d2 / (t2 * t2);
double h2_d2 = 1.0 / t2;

// HK from H and MSQ (kinematic shape factor)
// HK2, HK2_H, HK2_MSQ already available from ComputeKinematicParameters
// Chain to T, D, U:
double hk2_t2 = hk2_h * h2_t2;
double hk2_d2 = hk2_h * h2_d2;
double hk2_u2 = hk2_msq * msq2_u2;  // if compressibility
double hk2_ms = hk2_msq;             // Mach sensitivity

// RT = reybl * U * T
double rt2_t2 = reybl * u2;
double rt2_u2 = reybl * t2;
double rt2_re = u2 * t2;  // (if needed for RE sensitivity)

// M2 sensitivity (M2 = U^2 * freestream_msq_factor)
double m2_u2 = 2.0 * u2 * msq_fac;  // or from compressibility
double m2_ms = u2 * u2;              // MSQ sensitivity
```

Then chain ALL correlation derivatives through these primaries:

```
// Skin friction chains (Fortran BLVAR lines 912-950)
cf2_t2 = cf2_hk * hk2_t2 + cf2_rt * rt2_t2;
cf2_d2 = cf2_hk * hk2_d2;
cf2_u2 = cf2_hk * hk2_u2 + cf2_rt * rt2_u2 + cf2_msq * m2_u2;
cf2_ms = cf2_hk * hk2_ms + cf2_rt * rt2_ms + cf2_msq * m2_ms;
// Same for cf1 at station 1

// HS (energy shape) chains
hs2_t2 = hs2_hk * hk2_t2 + hs2_rt * rt2_t2;
hs2_d2 = hs2_hk * hk2_d2;
hs2_u2 = hs2_hk * hk2_u2 + hs2_rt * rt2_u2 + hs2_msq * m2_u2;

// HC (density thickness) chains
hc2_t2 = hc2_hk * hk2_t2;
hc2_d2 = hc2_hk * hk2_d2;
hc2_u2 = hc2_hk * hk2_u2 + hc2_msq * m2_u2;

// DI (dissipation) chains
di2_t2 = di2_hk * hk2_t2 + di2_rt * rt2_t2;
di2_d2 = di2_hk * hk2_d2;
di2_u2 = di2_hk * hk2_u2 + di2_rt * rt2_u2 + di2_msq * m2_u2;

// US (slip velocity) chains (Fortran BLVAR lines 822-831)
us2 = 0.5 * hs2 * (1.0 - (hk2 - 1.0) / (GBCON * h2));
us2_hs2 = 0.5 * (1.0 - (hk2 - 1.0) / (GBCON * h2));
us2_hk2 = 0.5 * hs2 * (-1.0 / (GBCON * h2));
us2_h2 = 0.5 * hs2 * (hk2 - 1.0) / (GBCON * h2 * h2);
// Then chain: us2_t2 = us2_hs2*hs2_t2 + us2_hk2*hk2_t2 + us2_h2*h2_t2
```

**Step 2: Compute UPW (upwinding) derivatives (Fortran BLDIF lines 1597-1643)**

The current code computes UPW value but NOT its derivatives. Add:

```
// Upwinding parameter and its full derivative chain
double hdcon = 5.0 * hupwt / (hk2 * hk2);
double hd_hk2 = -hdcon * 2.0 / hk2;
// (wake: hdcon = hupwt / hk2^2, same hd_hk2 pattern)

double arg = Math.Abs((hk2 - 1.0) / (hk1 - 1.0));
double hl = Math.Log(arg);
double hl_hk1 = -1.0 / (hk1 - 1.0);
double hl_hk2 = 1.0 / (hk2 - 1.0);

double hlsq = Math.Min(hl * hl, 15.0);
double ehh = Math.Exp(-hlsq * hdcon);
double upw = 1.0 - 0.5 * ehh;
double upw_hl = ehh * hl * hdcon;
double upw_hd = 0.5 * ehh * hlsq;

double upw_hk1 = upw_hl * hl_hk1;
```

```
// Fortran BLSYS applies:
// BL1(k,l) = VS1(k,l) * [chain factor for variable l at station 1]
```
