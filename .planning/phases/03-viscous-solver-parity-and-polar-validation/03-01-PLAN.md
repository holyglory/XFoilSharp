    Create all viscous solver state models following the established Phase 2 mutable workspace pattern:

    1. **BoundaryLayerProfile.cs**: Per-station BL state record. Fields: theta (momentum thickness), dstar (displacement thickness), ctau (shear stress coefficient), massDefect (delta*xUe), edgeVelocity (Ue), hk (kinematic shape parameter), cf (skin friction), reTheta (Re based on theta), amplificationFactor (N), xi (BL arc-length coordinate). This is an immutable output record for result reporting.

    2. **BoundaryLayerSystemState.cs**: Mutable workspace matching XFoil's COMMON/BLPAR/ arrays. Constructor takes (int maxStations, int maxWakeStations). Arrays (all 0-based):
       - UEDG[maxStations, 2] (edge velocity, 2 sides)
       - THET[maxStations, 2] (theta)
       - DSTR[maxStations, 2] (delta*)
       - CTAU[maxStations, 2] (shear stress)
       - MASS[maxStations, 2] (mass defect)
       - XSSI[maxStations, 2] (BL arc-length xi)
       - ITRAN[2] (transition station index, per side)
       - IBLTE[2] (TE station index, per side)
       - NBL[2] (number of BL stations per side)
       - TINDEX[2] (interpolated transition xi location)
       - Properties: Converged, RmsResidual, MaxResidualLocation, Iteration
       - Method: InitializeForStationCounts(int side1, int side2, int wake)

    3. **ViscousNewtonSystem.cs**: Pre-allocated Newton system matching BLSOLV structure.
       - VA[3, 2, maxStations] (diagonal blocks)
       - VB[3, 2, maxStations] (sub-diagonal blocks)
       - VM[3, maxWake, maxStations] (mass defect influence column)
       - VDEL[3, 2, maxStations] (RHS / solution deltas)
       - VZ[3, 2] (TE coupling block)
       - ISYS[maxStations, 2] (BL station -> system line mapping)
       - NSYS (total system lines)

    4. **ViscousConvergenceInfo.cs**: Readonly record struct with RmsResidual, MaxResidual, MaxResidualStation, MaxResidualSide, RelaxationFactor, TrustRegionRadius, CL, CD, CM, Iteration.

    5. **ViscousAnalysisResult.cs**: Sealed class with LiftCoefficient, DragDecomposition, MomentCoefficient, TransitionInfo (upper, lower), BoundaryLayerProfiles (upper, lower, wake arrays), ConvergenceHistory (list of ViscousConvergenceInfo), Converged, Iterations.

    6. **DragDecomposition.cs**: Readonly record struct with CD, CDF, CDP, CDSurfaceCrossCheck, DiscrepancyMetric, TEBaseDrag, WaveDrag (nullable for M > 0.7).

    7. **TransitionInfo.cs**: Readonly record struct with XTransition (x/c), StationIndex, TransitionType (enum: Free, Forced), AmplificationFactorAtTransition, Converged.

    8. **PolarSweepMode.cs**: Enum with AlphaSweep (Type 1), CLSweep (Type 2), ReSweep (Type 3).

    9. **ViscousSolverMode.cs**: Enum with TrustRegion (default), XFoilRelaxation.

    10. **AnalysisSettings.cs**: Extend with new optional constructor parameters (all at end, all with defaults to preserve backward compatibility):
        - ViscousSolverMode viscousSolverMode = ViscousSolverMode.TrustRegion
        - double? forcedTransitionUpper = null (x/c or null for free)
        - double? forcedTransitionLower = null
        - bool useExtendedWake = true
        - bool useModernTransitionCorrections = true
        - int maxViscousIterations = 50
        - double viscousConvergenceTolerance = 1e-6
        - double? nCritUpper = null (per-surface N_crit for upper surface; when null, uses global CriticalAmplificationFactor -- per CONTEXT.md locked decision: "N_crit configurable per-analysis and optionally per-surface (upper/lower independently)")
        - double? nCritLower = null (per-surface N_crit for lower surface; when null, uses global CriticalAmplificationFactor)
        - bool usePostStallExtrapolation = false (Viterna-Corrigan post-stall extrapolation -- per CONTEXT.md locked decision: "Post-stall extrapolation (Viterna-Corrigan style) available as optional mode with explicit flag; default reports only converged results")

    Preserve ALL existing constructor parameters and properties. Add new ones as optional parameters after existing ones. Add corresponding readonly properties:
        - NCritUpper (double?) -- returns nCritUpper
        - NCritLower (double?) -- returns nCritLower
        - UsePostStallExtrapolation (bool) -- returns usePostStallExtrapolation
