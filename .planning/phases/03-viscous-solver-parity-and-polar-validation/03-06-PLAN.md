  <behavior>
    - ComputeDrag for converged BL state: CD > 0, CDF > 0, CDP may be positive or negative
    - Squire-Young: CD = 2*theta_wake * (Ue_wake/Qinf)^(0.5*(5+H_wake))
    - CDF: integral of Cf * Ue/Qinf along both surfaces, projected onto freestream direction
    - CDP = CD - CDF
    - TE base drag: ~0.5 * (TE_gap/chord) * delta_Cp_base, small contribution
    - Surface cross-check: independent integration of friction + pressure along surface within 5% of wake CD
    - Compressibility transform (Karman-Tsien) applied to wake Ue before Squire-Young
    - For M > 0.7 with Lock wave drag enabled: WaveDrag estimate (nullable, non-null)
    - For M < 0.7: WaveDrag is null
  </behavior>
  <action>
    Create DragCalculator as a public static class in Services/.

    1. **CDCALC** (xfoil.f:1162-1198): ComputeDrag
       ```
       ComputeDrag(
           BoundaryLayerSystemState blState,
           LinearVortexPanelState panel,
           double qinf,
           double alfa,
           double machNumber,
           double teGap,
           bool useExtendedWake,
           bool useLockWaveDrag)
       -> DragDecomposition
       ```

       Implementation:
       - **Wake-end quantities**: theta_wake = THET[NBL[1]-1, 1], Ue_wake, H_wake = DSTR/THET at wake end
       - **Compressibility correction on Ue_wake**: Apply Karman-Tsien (TKLAM) to wake Ue: UE_comp = UE * (1 - TKLAM) / (1 - TKLAM * (UE/QINF)^2)
       - **Squire-Young extrapolation**: CD = 2.0 * theta_wake * (Ue_comp/QINF)^(0.5*(5.0 + H_wake))
       - **Skin friction drag** (IntegrateSkinFriction): Integrate Cf along both surfaces. At each station, CDF_local = Cf * (Ue/QINF)^2 * cos(theta_station - alfa) * ds/chord. Sum over all surface stations.
       - **Pressure drag**: CDP = CD - CDF
       - **Surface cross-check** (independent integration): Integrate (Cf * cos(angle) + Cp * sin(angle)) along both surfaces. This should agree with wake CD to within ~5%.
       - **TE base drag**: CD_base = 0.5 * (teGap / chord) * delta_Cp_base, where delta_Cp_base is estimated from TE velocity jump.
       - **Lock wave drag** (for M > 0.7, per user decision): Basic estimate using critical Mach from airfoil thickness ratio. CD_wave = K * (M - M_crit)^4 where K is calibrated. This is a Claude discretion item per CONTEXT.md. Only computed when useLockWaveDrag=true AND M > 0.7.
       - **Discrepancy metric**: |CD_wake - CD_surface| / CD_wake -- how well the two independent drag methods agree.

    2. **IntegrateSkinFriction**: Private helper that walks both surfaces integrating Cf.

    3. **ComputeTEBaseDrag**: Private helper for TE gap contribution.

    Wire into ViscousSolverEngine.SolveViscous (Plan 05): after Newton convergence, call DragCalculator.ComputeDrag and store result in ViscousAnalysisResult.DragDecomposition. If Plan 05 left a placeholder for drag, fill it in.
