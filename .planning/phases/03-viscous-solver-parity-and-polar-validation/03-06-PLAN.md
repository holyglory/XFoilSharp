    1. **CDCALC** (xfoil.f:1162-1198): ComputeDrag
       ```
       ComputeDrag(
           BoundaryLayerSystemState blState,
           LinearVortexPanelState panel,
           double qinf,
           double alfa,
           double machNumber,
           double teGap,
           bool useExtendedWake,
           bool useLockWaveDrag)
       -> DragDecomposition
       ```

       Implementation:
       - **Extended wake marching** (when useExtendedWake=true, the default per CONTEXT.md):
         Before computing Squire-Young CD, march wake stations forward from the last computed wake station.
         At each additional station, compute theta and H using the turbulent/wake BL equations (same as in BLSYS ITYP=3 mode).
         Continue marching until the convergence criterion is met: |dtheta/dxi| < epsilon (epsilon = 1e-6 or similar small value).
         Use the converged wake-end theta, Ue, H for the Squire-Young formula below.
         When useExtendedWake=false, skip extended marching and use XFoil's existing fixed-length wake end quantities directly.
       - **Wake-end quantities**: theta_wake = THET[NBL[1]-1, 1], Ue_wake, H_wake = DSTR/THET at wake end (or extended wake end if marching was performed)
       - **Compressibility correction on Ue_wake**: Apply Karman-Tsien (TKLAM) to wake Ue: UE_comp = UE * (1 - TKLAM) / (1 - TKLAM * (UE/QINF)^2)
       - **Squire-Young extrapolation**: CD = 2.0 * theta_wake * (Ue_comp/QINF)^(0.5*(5.0 + H_wake))
       - **Skin friction drag** (IntegrateSkinFriction): Integrate Cf along both surfaces. At each station, CDF_local = Cf * (Ue/QINF)^2 * cos(theta_station - alfa) * ds/chord. Sum over all surface stations.
       - **Pressure drag**: CDP = CD - CDF
       - **Surface cross-check** (independent integration): Integrate (Cf * cos(angle) + Cp * sin(angle)) along both surfaces. This should agree with wake CD to within ~5%.
       - **TE base drag**: CD_base = 0.5 * (teGap / chord) * delta_Cp_base, where delta_Cp_base is estimated from TE velocity jump.
       - **Lock wave drag** (for M > 0.7, per user decision): Basic estimate using critical Mach from airfoil thickness ratio. CD_wave = K * (M - M_crit)^4 where K is calibrated. This is a Claude discretion item per CONTEXT.md. Only computed when useLockWaveDrag=true AND M > 0.7.
       - **Discrepancy metric**: |CD_wake - CD_surface| / CD_wake -- how well the two independent drag methods agree.

       ExtrapolatePostStall(
           double alpha,                // current angle of attack
           double lastConvergedAlpha,   // last converged point's alpha

       Implementation (Viterna-Corrigan model):
       - CL_max from last converged point
       - For alpha > alpha_stall:
         CL = CL_max * sin(2*alpha) / (2 * sin(alpha_stall)) + CD_max * cos(alpha)^2 / sin(alpha)
         CD = CD_max * sin(alpha)^2 + CL_max^2 / (pi * AR)
       - This is a smooth extrapolation from the last converged point to the flat-plate limit at 90 degrees
       - Calibration parameters (CD_max, blending) are Claude's discretion per CONTEXT.md
