<objective>
Port the Newton iteration core: SETBL (global Newton system assembly), UPDATE (Newton step with relaxation), and VISCAL (outer iteration loop). This is the heart of the viscous solver.

Purpose: The outer Newton loop is what actually converges the viscous/inviscid coupled system. SETBL marches through all BL stations building the block-tridiagonal system, BLSOLV solves it, and UPDATE applies the step with careful relaxation. Without this, there is no viscous solver.

Output: ViscousNewtonAssembler, ViscousNewtonUpdater, ViscousSolverEngine, integration tests showing convergence.
</objective>

  <action>
    **ViscousNewtonAssembler.cs** (public static class):

    Port SETBL from xbl.f (lines ~1-500). BuildNewtonSystem method:

    1. **Marching order** (critical -- must match Fortran exactly):
       - IS=0 (side 1): IBL = 2 to IBLTE[0], calling BLSYS for each station pair
       - IS=1 (side 2): IBL = 2 to IBLTE[1], calling BLSYS for each station pair
       - Wake: IBL = IBLTE[1]+1 to NBL[1], calling BLSYS (ITYP=3)
       - At the first wake point, call TESYS instead of BLSYS

    2. **Station type dispatch** (ITYP):
       - Before ITRAN: ITYP=1 (laminar), using DAMPL for amplification
       - After ITRAN: ITYP=2 (turbulent), using Ctau equation
       - At ITRAN: ITYP=2 with TRDIF for transition interval
       - Wake: ITYP=3

    3. **DIJ coupling**: After computing local VA/VB/VDEL from BLSYS, add the edge velocity coupling terms from the DIJ matrix. For each station i, the mass defect at station j influences Ue at i via DIJ[i,j]. This adds off-diagonal terms to VM (mass influence column).

    4. **System line mapping**: Use ISYS to map BL station indices to the global system line numbers.

    5. **Compute RMSBL**: RMS of all residuals in VDEL before solving.

    **ViscousNewtonUpdater.cs** (public static class):

    Port UPDATE from xbl.f. ApplyNewtonUpdate method:

    1. **RLXBL relaxation** (xbl.f UPDATE):
       - After BLSOLV produces VDEL (Newton deltas), apply relaxation
       - Compute RLXBL = minimum relaxation factor across all stations/variables
       - For each variable: DMAX = max allowed change
         - Log(delta*): DHI = 1.5 (max increase), DLO = -0.5 (max decrease)
         - Theta: similar limits
         - Ctau: clamp to [0, 0.3]
       - RLXBL = min(1.0, min over all stations of DMAX/|delta|)

    2. **Apply update**:
       - THET += RLXBL * VDEL(theta component)
       - MASS += RLXBL * VDEL(mass defect component)
       - CTAU += RLXBL * VDEL(Ctau component)
       - Recompute DSTR = MASS / UE and H = DSTR / THET

    3. **Variable limiting**:
       - Ctau clamped to [0, 0.25] per user decision
       - No negative Ue islands
       - Hk >= 1.02 (laminar), Hk >= 1.00005 (wake)
       - Theta > 0

    4. **Trust-region mode** (modern default per user decision):
       When ViscousSolverMode == TrustRegion:
       - Instead of RLXBL, use Levenberg-Marquardt trust-region radius
       - Initial radius = 1.0
       - If step reduces residual: expand by 2.0, accept step
       - If step increases residual: shrink by 0.5, reject and retry
       - 3 consecutive shrinks = non-convergence for this iteration

    5. **XFoil-relaxation mode** (for Phase 4 verification):
       When ViscousSolverMode == XFoilRelaxation:
       - Use exact RLXBL logic from Fortran with DHI=1.5, DLO=-0.5

    Both modes: record per-iteration convergence info (RMS, max residual, location, relaxation/radius).

    **Initialization sequence** (before iteration):
    1. Calculate wake trajectory (extend WakeGeometryGenerator or use existing)
    2. Set wake velocities from airfoil vorticity (QWCALC)
    3. Set velocities for initial alpha (QISET from basis solutions)
    4. Set up BL station <-> panel pointers (STFIND, IBLPAN, XICALC, IBLSYS)
    5. Set inviscid BL edge velocity (UICALC)
    6. Initialize BL variables from inviscid Ue (upgrade ViscousStateEstimator -- march with inviscid Ue to get initial theta, delta* profiles; Blasius fallback if march fails)
    7. Build DIJ source influence matrix (QDCALC via InfluenceMatrixBuilder)

    **Newton iteration** (VISCAL main loop):
    ```
    for iter = 0 to maxViscousIterations:
        a. Build Newton system (ViscousNewtonAssembler.BuildNewtonSystem -> SETBL)
        b. Record pre-solve RMSBL
        c. Solve block-tridiagonal system (BlockTridiagonalSolver.Solve -> BLSOLV)
        d. Apply Newton update with relaxation (ViscousNewtonUpdater.ApplyNewtonUpdate -> UPDATE)
        e. If alpha-prescribed (Type 1):
           - Update Mach/Re from new CL (MRCL, COMSET)
        f. If CL-prescribed (Type 2):
           - Compute new alpha for target CL
           - Set new inviscid speeds (QISET, UICALC)
        g. Compute edge velocities from UEDG (QVFUE)
        h. Set GAM from QVIS (GAMQV)
        i. Relocate stagnation point (StagnationPointTracker.MoveStagnationPoint -> STMOVE)
        j. Calculate CL, CM (reuse LinearVortexInviscidSolver.IntegratePressureForces with viscous QVIS)
        k. Record iteration convergence info
        l. Check convergence: RMSBL < settings.ViscousConvergenceTolerance
           - If converged, break
    ```

    **Post-convergence**: Package results into ViscousAnalysisResult (CL, CD from CDCALC placeholder -- drag computed in Plan 06, CM, transition info, BL profiles, convergence history).

    **BL initialization** (step 6 above): Upgrade the existing ViscousStateEstimator to march BL variables from stagnation point using inviscid Ue. This replaces the surrogate's local estimation. The march uses:
    - Blasius similarity solution at stagnation for theta_0, delta*_0
    - Thwaites integral method to march theta along each surface
    - Transition detection via CheckTransition (N > Ncrit -> turbulent)
    - After transition, use turbulent correlations for continued march
    - This produces initial BL profiles close enough for Newton to converge
