Port PANGEN from xfoil.f lines 1613-2268 into a new `XFoilPanelGenerator` class. This is a substantial routine (~650 lines of Fortran) that implements XFoil's specific panel distribution algorithm.

**Overview of PANGEN algorithm:**
1. Take the buffer airfoil coordinates (XB[], YB[] with NB points from the input file)
2. Spline them using SPLIND
3. Find the LE location (minimum x via Newton iteration on the spline)
4. Set up a cosine-spacing base distribution with N panels
5. Apply refinement iterations to cluster panels near curvature peaks, LE, and TE
6. Evaluate the splined buffer coordinates at the new parameter positions to get X[], Y[]
7. Compute arc-length S[], spline derivatives XP[], YP[]
8. Set XLE, YLE, XTE, YTE, SLE, CHORD

**Method signature:** `static void Pangen(double[] xb, double[] yb, int nb, XFoilPanelState panel, int npan)`
- xb[], yb[]: input buffer airfoil coordinates (1-based)
- nb: number of input points
- panel: output XFoilPanelState to populate
- npan: desired number of panels

**Key implementation details:**

a. **LE detection** (lines ~1650-1690): Find the LE as the point on the splined airfoil with minimum x-coordinate. Uses Newton iteration: find s where dX/dS = 0. Store as SLE, XLE, YLE.

b. **Cosine base distribution** (lines ~1700-1750): Create a base cosine-spaced distribution from 0 to 1 on upper and lower surfaces separately. The distribution parameter goes from TE (s=0) around the LE (s=0.5) to TE (s=1.0).

c. **Curvature-based refinement** (lines ~1760-1900): Iteratively adjust the base distribution to cluster panels where curvature is high. Uses CVPAR (curvature density parameter), CTERAT (TE ratio), CTRRAT (curvature ratio). XFoil defaults: NPAN (from XFOIL.INC defaults, typically 140-160), CVPAR=1.0, CTERAT=0.15, CTRRAT=0.2.

d. **Final interpolation** (lines ~1900-2000): Evaluate the splined buffer airfoil at the refined parameter positions. Populate panel.X[], panel.Y[] for i=1..N.

e. **Post-processing** (lines ~2000-2100): Compute S[] via SCALC, spline X and Y with SPLIND, set CHORD, XTE, YTE.

f. **SPLNXY** (spline.f lines 452-530): The PANGEN routine calls SPLNXY to compute arc-length-parameterized splines with true arc-length re-integration. Port this as a helper method within XFoilPanelGenerator or as a method in XFoilSpline.

**The routine also calls SINVRT** (spline.f lines 404-435) for inverse spline evaluation (given x, find s). Port this as well.

Port the entire routine line-by-line. The paneling defaults should match XFoil's XFOIL.INC defaults (NPAN=160, CVPAR=1.0, CTERAT=0.15, CTRRAT=0.2). Expose these as optional parameters with XFoil-default values.

Do NOT modify the existing PanelMeshGenerator.cs. The new XFoilPanelGenerator is a separate, parallel implementation for the parity path.

Add `// Fortran: xfoil.f line NNN` comments at key algorithm stages.
