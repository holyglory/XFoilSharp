- `public static void FitWithZeroSecondDerivativeBCs(double[] values, double[] derivatives, double[] parameters, int count)` -- fits cubic spline with natural (zero second derivative) end conditions. **ScaledPivotLuSolver.cs**: Static utility class.

- `public static void Decompose(double[,] matrix, int[] pivotIndices, int size)` -- in-place LU decomposition with Crout's method and scaled partial pivoting. 0-based arrays. Modifies matrix in-place to contain L and U factors. Populates pivotIndices with row swap history.
  - Step 1: For each row, compute a scaling factor = 1.0 / max(|row elements|). Store in a temporary scaling array.
  - Step 2: For each column (Crout's outer loop): compute column of U (upper triangular) and row of L (lower triangular) simultaneously.
  - Step 3: Pivot selection: for each candidate row below the diagonal, compute scaled magnitude = scalingFactor[row] * |element|. Choose the row with the largest scaled magnitude. This is what makes it "scaled" pivoting -- it prevents rows with large absolute values from dominating.
  - Step 4: If pivot row differs from current column, swap rows in both the matrix and scaling array.
  - Step 5: Store pivot index and divide column below diagonal by pivot.

- `public static void BackSubstitute(double[,] luMatrix, int[] pivotIndices, double[] rhs, int size)` -- solves the system using LU factors. 0-based arrays. Modifies rhs in-place with the solution.
  - Forward substitution using L factor (with pivot unscrambling from pivotIndices).
  - Backward substitution using U factor.

Do NOT modify the existing DenseLinearSystemSolver.cs.
