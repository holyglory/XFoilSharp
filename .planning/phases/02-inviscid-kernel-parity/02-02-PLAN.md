**ComputeNormals(LinearVortexPanelState panel)**: Computes outward unit normal vectors at each node.
- Uses ParametricSpline.FitSegmented to compute spline derivatives of X and Y with respect to arc length
- For each node i: normalX[i] = yDeriv[i] / magnitude, normalY[i] = -xDeriv[i] / magnitude (outward normal from tangent rotation)
- Corner handling: detect corners where the arc-length spacing ratio between consecutive intervals exceeds 3x. At corners, average the normals from the two adjacent smooth segments. The algorithm: if |ds[i] - ds[i-1]| / min(ds[i], ds[i-1]) > 3, then node i is a corner point -- average adjacent normals.
- Populates panel.NormalX[], panel.NormalY[]

**ComputePanelAngles(LinearVortexPanelState panel)**: Computes panel angles at each node.
- For each panel between nodes i and i+1: compute tangent direction from node positions
- PanelAngle[i] = atan2(dy, -dx) where dx = X[i+1]-X[i], dy = Y[i+1]-Y[i]. Note the negation on dx -- this is the XFoil convention where panel angle is measured from the inward normal direction.
- For the last node (wrapping panel from node N-1 back to node 0), use the closing TE panel direction.
- Populates panel.PanelAngle[]

**ComputeTrailingEdgeGeometry(LinearVortexPanelState panel, InviscidSolverState state)**: Analyzes the TE gap.
- TrailingEdgeGap = distance between node 0 and node N-1 (first and last airfoil nodes)
- IsSharpTrailingEdge = TrailingEdgeGap < 0.0001 * Chord
- TrailingEdgeAngleNormal and TrailingEdgeAngleStreamwise: these are direction cosines of the TE gap vector decomposed into normal and streamwise components. Computed from the spline derivatives (XDerivative, YDerivative) at the first and last nodes. Specifically:
  - Let dx = X[0] - X[N-1], dy = Y[0] - Y[N-1]
  - ANTE = -(XDerivative[0]*dy - YDerivative[0]*dx + XDerivative[N-1]*dy - YDerivative[N-1]*dx) / 2
  - ASTE =  (XDerivative[0]*dx + YDerivative[0]*dy + XDerivative[N-1]*dx + YDerivative[N-1]*dy) / 2
  - These give the area-based decomposition of the TE gap into normal and streamwise components.
- If sharp: set TE bisector ratios SCS=ANTE/DSTE, SDS=ASTE/DSTE = 0.0 (or compute from spline tangent directions at TE nodes)
- Populates state.TrailingEdgeGap, state.TrailingEdgeAngleNormal, state.TrailingEdgeAngleStreamwise, state.IsSharpTrailingEdge

**ComputeCompressibilityParameters**: Returns a struct/record with Karman-Tsien parameters.
- Input: Mach number
- BETA = sqrt(1 - M^2)
- BFAC = M^2 / (2 * (1 + BETA))
- At M=0: BETA=1, BFAC=0, correction degenerates to incompressible Cp

**ContinuousAtan2(double y, double x, double referenceAngle)**: Returns atan2(y,x) with branch-cut continuation relative to referenceAngle. Adjusts by +/-2*PI to stay within PI of the reference. This prevents discontinuous jumps in angle-based quantities.

**ComputeInfluenceAt**: Computes the streamfunction PSI and all sensitivity arrays at a single field point due to all panels, the TE panel, and the freestream.

Signature:
```csharp
public static (double psi, double psiNormalDerivative) ComputeInfluenceAt(
    int fieldNodeIndex,
    double fieldX, double fieldY,
    double fieldNormalX, double fieldNormalY,
    bool computeGeometricSensitivities,
    bool includeSourceTerms,
    LinearVortexPanelState panel,
    InviscidSolverState state,
    double freestreamSpeed,
    double angleOfAttackRadians)
```

Parameters:
- fieldNodeIndex: which node the field point is (used for self-influence detection). Use -1 for non-node points.
- fieldX/Y: field point coordinates
- fieldNormalX/Y: outward normal at field point (for tangential velocity computation)
- computeGeometricSensitivities: if false, skip GEOLIN-related terms (per research: set false for direct analysis)
- includeSourceTerms: if true, compute source influence (needed for GGCALC assembly)
- panel, state: geometry and workspace
- freestreamSpeed, angleOfAttackRadians: freestream conditions

The method writes to state workspace arrays:
- state.StreamfunctionVortexSensitivity[j] = dPSI/dGamma_j (DZDG in XFoil)
- state.StreamfunctionSourceSensitivity[j] = dPSI/dSigma_j (DZDM in XFoil)
- state.VelocityVortexSensitivity[j] = dQ_tangential/dGamma_j (DQDG in XFoil)
- state.VelocitySourceSensitivity[j] = dQ_tangential/dSigma_j (DQDM in XFoil)

Returns (psi, psiNormalDerivative).

**Algorithm structure (implement as clean C#, NOT line-by-line Fortran):**

1. **Initialize**: Zero all sensitivity arrays. Set scaling constants: quarterOverPi = 1/(4*PI), halfOverPi = 1/(2*PI). Compute a small-panel threshold: smallPanelThreshold = (panel.ArcLength[N-1] - panel.ArcLength[0]) * 1e-5.

2. **Main panel loop** (for each panel j = 0 to NodeCount-2):
   - Get next node index jp = j + 1
   - Compute panel tangent: sx = (X[jp]-X[j])/panelLength, sy = (Y[jp]-Y[j])/panelLength, panelLength = distance(j, jp)
   - Skip degenerate panels where panelLength < smallPanelThreshold
   - Transform field point to panel-local coordinates: x1, x2 (along panel), yy (perpendicular to panel)
   - Compute log(r^2) terms: g1 = ln(x1^2 + yy^2), g2 = ln(x2^2 + yy^2) with self-influence protection (set g1=0 when fieldNodeIndex==j, g2=0 when fieldNodeIndex==jp, to avoid log(0))
   - Compute atan2 terms: t1, t2 with self-influence protection (set to -halfOverPi when on own node) and sign-reflection handling for airfoil surface points
   - **Vortex contribution** (linear vorticity): Compute sum integral psis = 0.5*x1*g1 - 0.5*x2*g2 + x2 - x1 + yy*(t1-t2) and difference integral psid = ((x1+x2)*psis + 0.5*(r2sq*g2 - r1sq*g1 + x1^2 - x2^2)) / (x1 - x2). Accumulate PSI += quarterOverPi * (psis*(gamma[jp]+gamma[j]) + psid*(gamma[jp]-gamma[j])). Accumulate sensitivity: dzdg[j] += quarterOverPi*(psis-psid), dzdg[jp] += quarterOverPi*(psis+psid).
   - **Source contribution** (if includeSourceTerms): Two half-panel quadratic source integrals. Split panel at midpoint x0=(x1+x2)/2. Compute psum/pdif for each half. Accumulate source PSI and source sensitivity arrays.
   - **Tangential velocity** (normal projection of velocity derivatives): Accumulate dqdg[] and dqdm[] arrays for velocity derivatives.

3. **TE panel**: After main loop, compute TE panel contribution from gap geometry. TrailingEdgeVortexStrength = SCS*dGamma + SDS*dSigma contributions (where SCS = ANTE/DSTE, SDS = ASTE/DSTE from the TE geometry). Add to PSI and sensitivity arrays.

4. **Freestream**: Add PSI += freestreamSpeed * (cos(alpha)*fieldY - sin(alpha)*fieldX).

5. **Sign reflection**: The SGN flag from XFoil ensures correct atan2 branch handling. For airfoil surface points (fieldNodeIndex < NodeCount), determine which side of the panel the field point is on and reflect if needed. For off-body points, no reflection needed.

Key numerical care:
- Use the sign-reflection trick (SGN in XFoil) to keep atan2 values well-behaved
- Self-influence: when the field point IS a panel endpoint, the log and atan terms degenerate -- set them to known limiting values instead of computing 0/0
- Degenerate panels (nearly zero length) should be skipped entirely
