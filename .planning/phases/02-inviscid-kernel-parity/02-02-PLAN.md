Create a new static class `XFoilGeometry` in Services directory with these methods, all ported line-by-line from Fortran:

**NCALC** (xpanel.f lines 51-84): Computes outward normal vectors from spline derivatives.
- Signature: `static void Ncalc(double[] x, double[] y, double[] s, int n, double[] nx, double[] ny)`
- Calls `XFoilSpline.Segspl` to compute XP[], YP[] spline derivatives
- Then normalizes: nx[i] = yp[i]/sqrt(xp^2+yp^2), ny[i] = -xp[i]/sqrt(xp^2+yp^2)
- Averages normals at corner points (lines 70-82): detects corners where arc-length spacing ratio > 3x, averages adjacent normals

**APCALC** (xpanel.f lines 22-48): Computes panel angles.
- Signature: `static void Apcalc(XFoilPanelState panel)`
- For each panel i=1..N-1: APANEL[i] = atan2(panel.SX, -panel.SY) where SX/SY are tangent components from adjacent nodes
- TE panel (i=N to 1 wrap): uses the TE-closing panel direction
- Note the swapped atan2 arguments -- this is intentional in XFoil

**TECALC** (xfoil.f lines 2270-2308): Computes TE gap geometry.
- Signature: `static void Tecalc(XFoilPanelState panel, XFoilInviscidState state)`
- DSTE = sqrt((X[1]-X[N])^2 + (Y[1]-Y[N])^2)
- ANTE = area-based TE gap component using XP/YP derivatives at nodes 1 and N
- ASTE = similar for the other component
- SHARP = DSTE < 0.0001 * CHORD
- Compute SCS = ANTE/DSTE, SDS = ASTE/DSTE (with zero-protection when SHARP)

**COMSET** (xfoil.f lines 1019-1044): Computes compressibility parameters.
- Signature: `static void Comset(XFoilInviscidState state, double minf)`
- Computes BETA = sqrt(1-MINF^2), BFAC for Karman-Tsien
- Computes TKLAM, TKL_MSQ, CPSTAR, QSTAR
- For MINF=0: TKLAM=0, TKL_MSQ=0, CPSTAR=-999, QSTAR=999

**ATANC** (xutils.f lines 68-107): Branch-cut-crossing atan2 continuation.
- Signature: `static double Atanc(double y, double x, double thold)`
- Returns atan2(y,x) with branch-cut continuation relative to thold

All methods use 1-based arrays. Add `// Fortran: {file} line NNN` comments.

Port PSILIN (xpanel.f lines 87-476) as a static method in a new `XFoilPsilin` class.

Signature: `static void Psilin(int i, double xi, double yi, double nxi, double nyi, out double psi, out double psi_ni, bool geolin, bool siglin, XFoilPanelState panel, XFoilInviscidState state)`

This is the largest and most critical routine to port. Follow these steps exactly:

1. **Initialize** (lines 109-140): Zero out DZDG[], DZDN[], DZDM[], DQDG[], DQDM[], PSI=0, PSI_NI=0, workspace scalars.
   - Compute SEPS = (S[N]-S[1]) * 1.0E-5 for degenerate panel detection.
   - Set QOPI = 1.0/(4*PI), HOPI = 1.0/(2*PI).

2. **Main panel loop** (lines 167-400): For each panel JO = 1..N:
   - Compute JP = JO+1 (next node, wrapping N+1 -> 1 is not needed since JO goes to N and JP=N+1 uses node 1 for TE)
   - Actually in XFoil, JP = JO+1 and the arrays go up to N, with special handling. Check: the loop goes JO=1 to N, JP=JO+1. When JO=N, JP=N+1 which wraps to node 1 (the TE panel closing the airfoil). The Fortran code handles this via the DO 25 loop with JO=1,N and JP=JO+1.

   For each panel:
   a. **Local coordinate transform** (lines 170-185): SX, SY (unit tangent), DSO (panel length), transform field point (XI,YI) to local coords (X1, X2, YY).
   b. **Log/atan terms** (lines 198-235): Compute RS1, RS2 (squared distances), G1=ln(RS1), G2=ln(RS2), T1=atan2(X1*YY, ...), T2=atan2(X2*YY, ...) with self-influence and SGN reflection handling.
   c. **Source contribution** (lines 237-332, only if SIGLIN): Two half-panel quadratic source integrals. Compute midpoint X0, then PSUM/PDIF for 1-0 half and 0-2 half. Accumulate DZDM[JO], DZDM[JP], PSI source terms.
   d. **Vortex contribution** (lines 336-372): Linear vorticity PSIS (sum integral) and PSID (difference integral). Accumulate PSI, DZDG[JO], DZDG[JP].
   e. **Tangential velocity derivatives** (lines 363-400): PSI_NI terms from DQDG[], DQDM[] using normal projection.

3. **TE panel** (lines 409-450): After the main loop, handle the explicit TE panel. Compute GAMTE and SIGTE from TE gap geometry (SCS, SDS ratios from TECALC). Add their contributions to PSI and derivative arrays.

4. **Freestream** (lines 451-460): Add PSI += QINF*(COSA*YI - SINA*XI) and corresponding derivatives.

5. **Skip LIMAGE branch** (lines 460+): Set LIMAGE=false per research recommendation. Do not port the ground-effect image system.

**Critical numerical details to preserve:**
- The SGN reflection flag (lines 189-197): For airfoil surface points (IO <= N), SGN is set based on which side of the panel YY falls on. For wake points, SGN = 1.0 always.
- Self-influence handling: When IO==JO, set G1=0, T1=-HOPI (half of atan2). When IO==JP, set G2=0, T2=-HOPI.
- DXINV = 1.0/(X1-X2) -- degenerate panel check: if DSO < SEPS, skip panel contribution.
- Preserve exact ordering of all additions and multiplications.

Add Fortran line-number comments throughout. This method is ~300 lines of C# -- it MUST match the Fortran operation-by-operation.
