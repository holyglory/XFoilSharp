Create a new `XFoilInviscidSolver` class in Services directory. This is the top-level inviscid solver that orchestrates the full analysis chain. Port these routines as methods:

**GGCALC** (xpanel.f lines 976-1111):
- Signature: `static void Ggcalc(XFoilPanelState panel, XFoilInviscidState state, double qinf)`
- For each node I=1..N: call Psilin(I, X[I], Y[I], NX[I], NY[I], ..., GEOLIN=false, SIGLIN=true)
- Store: AIJ[I,J] = DZDG[J], BIJ[I,J] = -DZDM[J] for J=1..N
- Set AIJ[I,N+1] = -1.0 (internal streamfunction column)
- Set RHS: GAMU[I,1] = -QINF*Y[I], GAMU[I,2] = QINF*X[I]
- Standard Kutta: AIJ[N+1,1] = 1, AIJ[N+1,N] = 1 (GAM at node 1 + GAM at node N = 0)
- GAMU[N+1,1] = 0, GAMU[N+1,2] = 0
- **Sharp TE override** (lines 1027-1080): If SHARP, replace row N (not N+1) with internal bisector zero-velocity condition. Compute bisector angle, place control point slightly inside TE, call Psilin at that point for DQDG, set AIJ[N,J] = DQDG[J].
- LU-factor: Ludcmp(IQX, N+1, AIJ, AIJPIV)
- Solve both RHS: extract columns of GAMU into temp arrays, call Baksub for each, store back
- Copy to QINVU: QINVU[I,1] and QINVU[I,2] from GAMU basis solutions
- Set LGAMU=true, LQAIJ=true

**SPECAL** (xoper.f lines 2429-2513):
- Signature: `static void Specal(double alfa, XFoilPanelState panel, XFoilInviscidState state, double qinf, double minf)`
- If !LGAMU: call Ggcalc first
- Superimpose: GAM[I] = cos(alfa)*GAMU[I,1] + sin(alfa)*GAMU[I,2] for I=1..N+1
- Compute GAM_A[I] (dGAM/dAlfa)
- PSIO = GAM[N+1] (internal streamfunction value from N+1 row)
- Call Tecalc to update GAMTE, SIGTE
- Call Qiset
- Newton iteration on CL-Mach coupling (MRCL + COMSET + CLCALC loop, lines 2470-2510). For M=0 this converges in 1 iteration.
- Store ALFA, COSA=cos(alfa), SINA=sin(alfa) in state

**SPECCL** (xoper.f lines 2516+):
- Signature: `static void Speccl(double clspec, XFoilPanelState panel, XFoilInviscidState state, double qinf, double minf)`
- Newton iteration: guess alfa, call Specal, use CL_ALF to update alfa until CL matches clspec

**QISET** (xpanel.f lines 1578-1595):
- Signature: `static void Qiset(XFoilPanelState panel, XFoilInviscidState state, double alfa)`
- QINV[I] = cos(alfa)*QINVU[I,1] + sin(alfa)*QINVU[I,2] for I=1..N+NW
- QINV_A[I] = -sin(alfa)*QINVU[I,1] + cos(alfa)*QINVU[I,2]

**CLCALC** (xfoil.f lines 1078-1158):
- Signature: `static XFoilInviscidResult Clcalc(XFoilPanelState panel, XFoilInviscidState state, double alfa, double minf, double qinf, double xcmref, double ycmref)`
- Compute BETA = sqrt(1-MINF^2), BFAC = MINF^2/(2*(1+BETA))
- For each panel I to I+1: compute KT-corrected Cp, integrate for CL, CDP, CM
- **Critical:** The CM formula includes the DG*DX/12 and DG*DY/12 second-order correction terms (research Pattern 4). This is what the current C# code misses.
- Also compute CL_ALF and CL_MSQ analytically alongside the integration
- Return XFoilInviscidResult with CL, CM, CDP, CL_ALF, CL_MSQ

**CPCALC** (xfoil.f lines 1047-1075):
- Signature: `static void Cpcalc(int n, double[] q, double qinf, double minf, double[] cp)`
- Compute KT-corrected Cp from surface speed: CPG = (1-(Q/QINF)^2) / (BETA + BFAC*(1-(Q/QINF)^2))

**MRCL** (xfoil.f lines 774-824):
- Signature: `static void Mrcl(double cls, double minf, out double minf_cl, int retyp, int matyp)`
- For MATYP=1 (fixed Mach): MINF_CL = 0
- For MATYP=2 or 3: compute Mach-CL dependency

Also add a high-level convenience method:
- `static XFoilInviscidResult AnalyzeInviscid(double[] xb, double[] yb, int nb, int npan, double alfaDeg, double minf)`
- Creates XFoilPanelState and XFoilInviscidState
- Calls Pangen -> Scalc -> Segspl/Ncalc/Apcalc -> Tecalc -> Comset -> Ggcalc -> Specal -> returns result
- This is the entry point for parity testing

Port all routines line-by-line from Fortran with operation-order preservation. Add Fortran line-number comments.
